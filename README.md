# dogPark

Historical hackathon prototype for a dog park social experience.

This repo is kept public as a portfolio case study, not as a claim that the original stack is the ideal current implementation. The value is the product idea, the speed of the original build, and the opportunity to revisit the concept with stronger architecture later.

## What It Shows

- Product thinking around a real-world social use case.
- A fast prototype shipped with Next.js, MDX, Tailwind, Formik, and a static export workflow.
- A useful candidate for a future v2 because the original vision is still clear.

## Chronology

- Original version: hackathon-era prototype.
- Restoration pass: dependency and build cleanup to keep the project reviewable.
- Future v2: rebuild the experience around modern routing, durable data, and a clearer user journey.

## Maintenance Status

Historical project, currently being restored. Recent work reduced stale dependency debt, but the remaining Next.js major-version upgrade is a real modernization task rather than a simple lockfile bump.
